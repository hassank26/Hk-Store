# 🛒 HK Store – Static Web Project

HK Store is a modern static e-commerce website built using HTML, CSS, and JavaScript. The project demonstrates a simple online shopping experience with a clean and user-friendly interface.

## 🚀 Features

* 🛍️ Browse products easily
* 🛒 Add to Cart functionality
* 📦 Dynamic cart panel (open/close)
* 🔢 Live cart item counter
* 🎨 Clean and responsive UI design
* ⚡ Smooth interactions using JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📁 Project Structure

```
Static Web Proj/
│── home.html       # Main webpage
│── style.css       # Styling file
│── script.js       # Handles cart and UI interactions
```

## 📸 Preview

This project simulates a simple online store where users can view products and add them to their cart.

## ▶️ How to Run

1. Clone or download this repository
2. Open `home.html` in your browser
3. Start exploring the store

## 🎯 Purpose

The main goal of this project is to:

* Improve frontend development skills
* Practice DOM manipulation
* Build a basic e-commerce interface
